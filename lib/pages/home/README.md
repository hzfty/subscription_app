Домашняя страница

1. Appbar с заголовокм "Ваш список" и кнопкой справа перехода к экарану подписок
2. В body должен лежать список из созданных карточек. Пока картчоек нет, должен быть соответствующий плейсхолдер
3. Внизу должен быть floatingbutton "+", при нажатии на котороый должно высвечиваться диалоговое окно с полями (модель карточки):
3.1 Название (обязательное поле)
3.2 Описание (необязательное поле)
3.3 Три вида чипов для выбора типа карточки:
3.3.1 Обычный (сервый цвет). Нажат по дефолту доступен всем
3.3.2 Плюс (фиолетовый цвет). Неактивен для пользователей с обычным доступом (только Плюс и премиум)
3.3.3 Премиум (золотой цвет). Неактивен для пользователей с обычным доступом (только Премиум)
3.4 Кнопки "Отмена", и "Создать", создающие карточку с выбранными данными (Тип отвечает за цвет обводки)
3.5 Зажатие карточки отображает простой диалог с предложенем удаления картчоки
